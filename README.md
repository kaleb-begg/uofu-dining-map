# U of U Dining Interactive Map

Interactive campus dining map using the University of Utah public Facilities GIS basemap, Leaflet, and a CSV data file.

## Files

- `index.html` — webpage structure
- `css/styles.css` — visual styling
- `js/app.js` — map logic, filters, analytics, CSV import/export
- `data/dining-data.csv` — dining location data

## GitHub Pages

Upload these files/folders to your repository root, then use Settings → Pages → Deploy from a branch → `main` → `/ (root)`.

If you update dining metrics later, edit or replace `data/dining-data.csv` with the same columns.
